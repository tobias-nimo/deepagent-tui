# Images

Attaching images to messages.

## Attaching

> Paste a file path, or drop a file into a terminal that yields a path. Attachments stack across edits; `Esc` clears any pending ones before sending.

## Supported formats

> Which file types the TUI recognizes, how unknown types are handled.

## Terminal support

> Which terminals reliably yield a path on drag-and-drop, and what to do when yours doesn't (paste the path manually).

## How attachments are sent

> Encoding, multimodal message shape, what the server sees.
